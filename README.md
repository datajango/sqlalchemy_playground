# SQL Alchemy Playground

- By Anthony Leotta
- 2/8/2022

## Overview

This playground uses Docker to run Postgres, TimescaleDB, MongoDb, MySQL, MariaDB, Percona, SQL Server.

## Setup

1. Run docker compose

    ```
    docker-compose up -d
    ```

1. Load Databases

## Topics

1. SQLAlchemy Core
1. SQLAlchemy ORM
1. Alembic section.
